# 바로영어 놀이터 데이터 출처

이 프로젝트의 영단어 학습 데이터는 공개 영어 빈도 자료와 영한 어휘 자료를 기반으로 정리합니다.

## 데이터 정제 기준

- 영문 표제어 중복 제거
- 고유명사, 숫자, 약어, 비속어 제외
- 대표 한국어 뜻 1개 사용
- 같은 의미권의 뜻은 오답 보기에서 제외
- 초등, 중등, 고등, TOEIC, TOEFL, 고급 어휘로 난이도 구분
- 20개 스테이지에 총 10,000개 이상의 단어 배치

TOEIC 및 TOEFL 표기는 학습 난이도 구간을 설명하기 위한 것이며 공식 출제기관의 공식 단어 목록을 의미하지 않습니다.
